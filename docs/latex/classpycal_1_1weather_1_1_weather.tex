\doxysubsection{Detailed Description}
\begin{DoxyVerb}cal Weather objects allow sampling weather parameters.

The weather parameter distributions are read from site-specific
cal weather files.  The files contain parameter distributions
for every UTC hour of the day, averaged over months.\end{DoxyVerb}
